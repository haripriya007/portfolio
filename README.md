## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build for Production

```bash
npm run build
```

Output goes to `dist/` — ready for GitHub Pages deployment.

## Deploy to GitHub Pages

1. Create a repo named `portfolio` on GitHub (public)
2. Push this branch and merge to `main`
3. Go to **Settings → Pages → Source: GitHub Actions**
4. The workflow at `.github/workflows/deploy.yml` handles everything automatically
5. Live at: `https://haripriya007.github.io/portfolio`
